# 📸 Clicksy

A modern social media platform focused on sharing moments, connecting with people, and engaging through interactive content.

---

## 🚀 Features

- 🔐 Authentication & Authorization
- 👤 User Profiles
- 🖼️ Image Uploads
- ❤️ Like & Comment System
- 📰 Dynamic Feed
- 👥 Follow / Unfollow Users
- 🔍 Explore & Search
- 📱 Responsive Design
- 🌙 Modern UI/UX
- ⚡ Real-time Updates

---

## 🛠️ Tech Stack

### Frontend

- React.js
- Vite
- Tailwind CSS
- Axios
- React Router

### Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Cookie Parser

## ⚙️ Environment Variables

Create a `.env` file inside the server folder.

```env
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
FRONTEND_URL=http://localhost:5173
```

## 📦 Installation

### Clone Repository

```bash
git clone https://github.com/dibyo20/Clicksy.git
```

### Install Dependencies

```bash
# Frontend
cd Frontend
npm install

# Backend
cd Backend
npm install
```

---

## ▶️ Run Locally

### Start Backend

```bash
cd Backend
npm run dev
```

### Start Frontend

```bash
cd Frontend
npm run dev
```
